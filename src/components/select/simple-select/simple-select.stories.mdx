# Simple Select

Select one of available options from the drop-down menu

## Overview 

Simple Select is a Carbon styled equivalent of HTML Select Element with optional Type To Search functionality.

## Guidance, hints and tips 

Always insert `Option` Components inside the `SimpleSelect`, analogous to the original `<select>` and `<option>` HTML Elements.

To enable Type To Search functionality you have to add `hasFilter` prop.
It will allow you to filter through the existing options leaving only these that matches the typed in text.

## Quick Start 

To use the SimpleSelect component you have to import two components: `<SimpleSelect />` and `<Option />`
from `carbon-react/lib/components/select`.
